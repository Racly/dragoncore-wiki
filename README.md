## 简介

龙之核心为一款 插件+Mod 型的多功能插件，提供了众多仅客户端Mod才能制作的功能

并通过插件来进行通讯，使得其可运行在任意bukkit服务端如spigot,catsetver,paper

插件使用版本： 1.12.2

Mod使用版本:   1.12.2-forge.2768及以上
